# DrawAI — черчение и геометрия по описанию нейросети

Веб-приложение, которое умеет рисовать чертежи и геометрические построения по
JSON-описанию, сгенерированному любой нейросетью (Claude / GPT / Gemini /
локальные LLM) из фотографии задания.

## Как это работает

1. Открываете вкладку **Промты**, выбираете тип задания (черчение / геометрия)
   и семейство нейросети, копируете готовый промт.
2. Открываете чат нейросети, вставляете промт и прикрепляете фотографию задания.
3. Нейросеть возвращает **JSON-DSL** — описание чертежа в миллиметрах.
4. Вставляете JSON во вкладку **Черчение** или **Геометрия**, нажимаете
   «Отрисовать» — приложение строит векторный чертёж и показывает его на листе
   с рамкой ГОСТ 2.301 и штампом ГОСТ 2.104 (для черчения) или на чистой
   плоскости с координатной сеткой 5 мм (для геометрии).
5. Экспорт в SVG/PNG, печать на бумаге A4/A3/A2/A1/A0.

## Запуск

Приложение полностью статическое — никакой сборки не нужно. Достаточно открыть
`index.html` в любом современном браузере (Chrome, Firefox, Safari, Edge).

```bash
# Простой локальный сервер (любой из):
python3 -m http.server 8080
# или
npx serve .
```

Затем открыть `http://localhost:8080`.

## Возможности

- **Три вкладки**: Черчение, Геометрия, Промты.
- **Форматы листа**: A4, A3, A2, A1, A0 — портретная и альбомная ориентации.
- **Штамп ГОСТ 2.104** (форма 1): наименование, обозначение, материал,
  масштаб, разработчик, проверяющий, учебное заведение.
- **Сетка 5 мм** с настройкой шага и прозрачности (0–100%), жирные линии
  каждые 10 клеток.
- **Инструменты**: панорама (✥), линейка (⟺) с автоизмерением расстояний и
  углов, транспортир (∠) для измерения углов по трём точкам, маркер точки (•).
  Привязка к узлам сетки (Shift отключает).
- **Зум** колесом мыши, кнопками `+`/`-` и «по листу».
- **Экспорт**: SVG (векторный, с фиксированным размером в мм для печати),
  PNG (4× растеризация), прямая печать.
- **Темы**: тёмная и светлая.

## DSL — формат данных

Корневой объект JSON:

```json
{
  "version": "1.0",
  "mode": "drawing" | "geometry",
  "format": "A3",
  "orientation": "landscape",
  "title": { ... },
  "viewport": { "width": 300, "height": 200 },
  "origin": { "x": 50, "y": 50 },
  "elements": [ ... ]
}
```

Полный список типов элементов с примерами — внутри файла `js/prompts.js`
(константа `DSL_REFERENCE`) и в готовых примерах: кнопка **«Пример»** на
вкладках Черчение/Геометрия.

Поддерживаемые элементы:

| Тип         | Назначение                                         |
| ----------- | -------------------------------------------------- |
| `line`      | Отрезок                                            |
| `polyline`  | Ломаная                                            |
| `polygon`   | Замкнутый многоугольник (с заливкой)               |
| `rect`      | Прямоугольник (можно скруглённый)                  |
| `circle`    | Окружность (опц. с осевыми)                        |
| `ellipse`   | Эллипс                                             |
| `arc`       | Дуга по центру, радиусу, углам                     |
| `path`      | Произвольная SVG-кривая                            |
| `text`      | Подпись                                            |
| `point`     | Точка с подписью (для геометрии)                   |
| `dimension` | Размер: linear, aligned, diameter, radius, angular |
| `angle`     | Дуга-маркер угла с подписью (без размерной линии)  |
| `hatch`     | Штриховка области под углом                        |
| `group`     | Группа с локальным смещением                       |

Стили линий: `solid` (видимый контур), `thin` (вспомогательная),
`dashed` (штриховая — невидимый контур), `dashdot` / `axis`
(штрихпунктирная — оси), `phantom` (двух-штрих-пунктирная).

## Структура проекта

```
index.html           Корневая разметка (3 вкладки)
css/styles.css       Стили и темы
js/app.js            Связывание UI и логики
js/renderer.js       Парсер DSL → SVG
js/formats.js        Форматы листа и штамп ГОСТ 2.104
js/tools.js          Pan/zoom, сетка, линейка, транспортир, экспорт
js/prompts.js        Шаблоны промтов для нейросетей
```
