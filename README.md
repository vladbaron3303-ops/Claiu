# Чертёжник — AI Drafting & Geometry

Браузерное приложение для отрисовки чертежей и геометрических задач по коду,
сгенерированному нейросетью. Не требует сборки: открой `index.html` в браузере.

## Как это работает

1. На вкладке **Промты** выбираешь категорию (черчение / планиметрия / стереометрия / справочник),
   семейство нейросети (ChatGPT / Claude / Gemini / DeepSeek / локальные) и нужный вариант.
2. Жмёшь **Скопировать** — промт попадает в буфер.
3. Заходишь к выбранной нейросети, прикрепляешь картинку задания и вставляешь промт.
4. Получаешь ответ — код на собственном языке **DSL Чертёжник**.
5. Возвращаешься в приложение, вставляешь код в нижнее поле на вкладке **Черчение**
   или **Геометрия** и жмёшь **Отрисовать** (или `Ctrl+Enter`).

## Вкладки

- **Черчение** — лист формата (A4/A3/A2/A1 в обеих ориентациях), ГОСТ-рамка
  и редактируемая основная надпись (Разработал, Проверил, Утвердил, Название, Масштаб, Литера,
  Масса, Лист/Листов, Дата, Школа/Группа, Материал). Координаты рисунка задаются в мм
  от левого верхнего угла рабочей зоны (внутри рамки, без штампа).
- **Геометрия** — бесконечная плоскость без рамки, ось Y по выбору (вверх — по-математически,
  или вниз — экранная), автоподгонка вида по команде `view`.
- **Промты** — библиотека готовых промтов с полной спецификацией DSL внутри.

## Инструменты во всех режимах

- Сетка 0,5 см (5 мм), регулируемая прозрачность.
- Курсор / Линейка (две точки — показывает Δx, Δy, длину и угол) / Рука (панорамирование).
- Колесо мыши — масштаб; Shift+drag или средняя кнопка — панорамирование.
- Толщина основных линий, общая прозрачность чертежа.
- Экспорт **PNG** и **SVG**.
- Горячие клавиши: `Ctrl+Enter` — перерисовать.

## DSL (миллиметры, одна команда на строку, `#` — комментарии)

```
# Системные
axis y-up                      # для геометрии — Y вверх
view x1 y1 x2 y2               # рамка отображения (геометрия)
layer main|thin|axis|hidden|dim|construction
style color=#000 width=0.5 dash=solid|dashed|dotted|dashdot|center
push / pop / reset
translate dx dy
rotate angle [cx cy]
scale sx [sy]
fill color=#000 alpha=0.5
nofill

# Примитивы
line x1 y1 x2 y2
polyline x1,y1 x2,y2 ...
polygon  x1,y1 x2,y2 ...
rect x y w h
circle cx cy r
ellipse cx cy rx ry [angle]
arc cx cy r startDeg endDeg
point x y
text x y "..." [size=3.5] [angle=0] [align=...] [baseline=...]

# Метки и размеры
mark x y "A"
right-angle x y dx dy [size=2.5]
tick / tick2 / tick3 x1 y1 x2 y2
dim-h / dim-v / dim-l x1 y1 x2 y2 [offset=10] [text="..."]
dim-r cx cy r [angle=45] [text="R10"]
dim-d cx cy r [angle=45] [text="⌀20"]
dim-a x1 y1 cx cy x2 y2 [radius=12] [text="30°"]

# Штриховка
hatch-rect x y w h [angle=45] [spacing=2]
hatch-poly x1,y1 x2,y2 ... [angle=45] [spacing=2]
```

## Запуск

```
# Любой статический сервер, например:
python3 -m http.server 8000
# затем открой http://localhost:8000
```

или просто открой `index.html` двойным кликом.

## Структура файлов

- `index.html` — каркас интерфейса
- `styles.css` — оформление
- `app.js` — вкладки, инструменты, рендер
- `dsl.js` — парсер и интерпретатор DSL
- `titleblock.js` — форматы листов и ГОСТ-штамп
- `prompts.js` — библиотека промтов
